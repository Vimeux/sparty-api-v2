# sparty-api-v2
V2 of the sparty API

## init project
```
npm install
```
or 
```
yarn
```
## Start project
```
npm dev
```
or
```
yarn dev
```

## Configuration

Set up an .env file with the port for the server, the DB connection information and the JWT key like this :

```
# SERVER
PORT=

# DATA BASE
DB_USERNAME=''
DB_HOST=''
DB_NAME=''
DB_PASSWORD=''
DB_PORT=

# JWT
JWT_SECRET=''
```
